# mp3-music-player
This is a mp3-music-player.

Code: Python.

已发布第一个版本，可以直接下载exe文件使用，无需安装。

2025年2月14日 更新：

功能列表：
1. 导入音乐，支持本地文件夹，选择文件夹导入。
2. 上一首
3. 暂停/播放
4. 下一首
5. 播放模式切换：单曲循环，顺序播放，随机播放
6. 收藏
7. 音量控制
8. 歌曲检索
